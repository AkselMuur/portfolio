# E‑Portfolio – Aksel Müür  
A personal web‑based portfolio showcasing my projects, skills and experience as a Junior Web Developer.  
The goal of this portfolio is to present my work in a clean, modern and accessible format while demonstrating my ability to build real, functioning web solutions.

---

## 🎯 Purpose  
This e‑portfolio was created to:

- present my development projects in a structured and professional way  
- demonstrate my skills in front‑end and back‑end development  
- provide live links to hosted projects  
- serve as a central hub for my CV, GitHub and contact information  

---

## 🛠️ Tech Stack  
- **HTML5** – semantic structure  
- **CSS3** – responsive layout, animations, styling  
- **JavaScript** – interactive components  
- **React** – component‑based UI (for some project pages)  
- **PHP** – backend logic for selected projects  
- **cPanel hosting** – deployment and server configuration  
- **GitHub** – version control and project management
